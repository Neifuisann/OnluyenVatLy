Static Mobile Navigation Test

This is a static test page for the mobile navigation menu.

Open this page on a mobile device or resize your browser window to test the mobile menu.

Test Instructions:

  1. Resize browser window to < 768px width
  2. Click the hamburger menu icon in the top right
  3. The menu should slide down from the top
  4. The hamburger icon should change to an X
  5. Clicking outside or on a link should close the menu
